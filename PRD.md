## Страница со списком постов /posts

Страница рендерит список постов из папки `public/posts/`.
Посты должны быть отсортированы по числу в начале названия файла поста от большего к меньшему (чем больше число, тем новее пост).
Посты до `19_consciousness.md` должны быть отделены заголовком "Старые посты".

Пост должен рендерится карточкой, на которой есть:

- Название поста взятое из h1 заголовка в начале файла поста
- Первой картинки из ссылки на картинку в файле поста
- Ссылки на публикации на других платформа взятые из `public/posts.json` где id - это имя файла поста из папки `public/posts/` без расширения `.md`.

Маппинг полей `public/posts.json` в названия ссылок:

- dzen - Дзен
- habr - Хабр
- pikabu - Пикабу
- vastrik - Вастрик

Нажатие на карточку поста открывает пост на отдельной странице `/posts/:id`, где id поста - название файла поста из папки `public/posts/` без расширения `.md`

## Страница поста /posts/:id

id поста - название файла поста из папки `public/posts/` без расширения `.md`
Рендер поста - рендер markdown-содержимого файла.
Вверху должны быть кнопка назад, ведущая на страницу всех постов `/posts`.

## Страница со списком заметок /notes

Страница рендерит список заметок из папки `public/notes/`.
Заметки должны быть отсортированы по числу в начале названия файла поста от меньшего к большему.

Заметка должен рендерится карточкой, на которой есть:

- Название, взятое из h1 заголовка в начале файла заметки

Нажатие на карточку заметки открывает её на отдельной странице `/notes/:id`, где id заметки - название файла заметки из папки `public/notes/` без расширения `.md`

## Страница заметки /notes/:id

id заметки - название файла заметки из папки `public/notes/` без расширения `.md`
Рендер заметки - рендер markdown-содержимого файла.
Вверху должны быть кнопка назад, ведущая на страницу всех заметок `/notes`.
